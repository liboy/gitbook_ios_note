# GitBook

## 安装
先行安装`Node.js`、`GitBook`、`GitBook Editor`、`Git`版本控制器
- 环境要求`NodeJS(v4.0.0及以上)`

- 通过`NPM`安装
```bash
npm install gitbook-cli -g
```
```bash
$ gitbook -V
CLI version: 2.3.2
GitBook version: 3.2.3
```
## 运行
* Clone 代码到本地并运行
```bash
git clone git@github.com:zhangjikai/gitbook-use.git
cd gitbook-use
gitbook install
gitbook serve
```
* 在浏览器中打开 `http://localhost:4000/` 进行访问
