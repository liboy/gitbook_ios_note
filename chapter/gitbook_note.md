# GitBook

## 运行
* 安装 GitBook
```bash
npm install gitbook-cli -g
```
* Clone 代码到本地并运行
```bash
git clone git@github.com:zhangjikai/gitbook-use.git
cd gitbook-use
gitbook install
gitbook serve
```
* 在浏览器中打开 `http://localhost:4000/` 进行访问
